Class: WireframeRenderer

Grape2D. WireframeRenderer

new WireframeRenderer(renderer)

Only renders the wireframe of (the bounding box of) Grape2D.Object2D, using another Grape2D.Renderer, such as Grape2D.CanvasRenderer. This class is a bridge, so in fact the renderer provided is the one that is being used.
