Class: Matrix

Grape2D. Matrix

new Matrix(aa, ab, ac, ba, bb, bc, ca, cb, cc)

Matrix defines a 3x3 matrix indicated to deal with 2D operations. If it's instantiated with no arguments then, it becomes the identity matrix.
