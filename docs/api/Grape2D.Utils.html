<static> requestAnimationFrame(w)

This is a semi-polyfill to request an animation frame, since it only changes this function in case it exists or don't. This may be a ugly and a bad pratice, bad it's only executed once and gives more abstraction to the developer. Once executed it will also place a polyfill into Grape2D.utils.cancelAnimationFrame. These polyfills are adapted from Erik's Moller requestAnimationFrame polyfill and fixed by Paul Irish and Tino Zijdel https://gist.github.com/paulirish/1579671
Parameters:
Name Type Description
w Object Window object. This object may or may not contain the requestAnimationFrame. If null then the global object window is taken.
