Class: CanvasRenderer

Grape2D. CanvasRenderer

new CanvasRenderer(options)

This is a specialization from Grape2D.Renderer. This renders objects to a canvas object, with a 2D context. This method is not hardware accelerated by default, however is the most stable and cross-browser.
