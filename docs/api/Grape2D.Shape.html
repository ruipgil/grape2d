Class: Shape

Grape2D. Shape

new Shape()

Shape is an abstract class that describes "physical", objects. The main objective of a Shape is to serve as a bounding box. It should play the main role when selecting the objects to render, when it comes to collision detection and/or to detect user interaction with an object.
